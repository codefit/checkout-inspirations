Checkout Pages - Demo

Checkout (Original)

Třísloupcový layout s fixní levou a pravou částí. Obsahuje slide widgety, které se vysunují zleva a překrývají střední část obsahu.

Funkce:
  • Slide widgety zleva
  • Fixní boční panely
  • Scrollovatelný střední obsah
  • Gradient overlay na pozadí
  • Kontaktní informace v levé části

Checkout V2

Dvousloupcový layout s formuláři vlevo a rekapitulací vpravo. Obsahuje accordion sekce a slide widgety pro výběr dopravy a platby.

Funkce:
  • Dvousloupcový layout (50/50)
  • Accordion pro platební údaje
  • Slide widgety zleva
  • Sticky pravá kolona
  • Firemní údaje checkbox
  • Bootstrap Icons
